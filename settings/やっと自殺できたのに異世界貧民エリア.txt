# 貧民エリア
・木造かそれ以下
・道は整備されてないが畑が多く、川も丘も森もある
・人口密度は高くて、どこを見ても人がいる
・貧しい冒険者用の宿もある

# 土地
・土地は貴族領という単位で管理されている
・ギルド第二十六領、シャーロット家第一領など
・領主のための食べ物をつくる

# 仕事
領主との関係
・どれだけ厳しく取り立てているかは家次第
・あまり厳しいと他領に逃げられるので、上手くコントロールが必要

衣食住
・仕事は衣食住で分かれている。三活分離と呼ぶ。
・領内を一手に引き受ける（衣担当は領民全員の衣をメンテする）
・食が一番面白くてしんどい。農業、狩猟、料理、保管。
・衣は衣服と狩猟用の武器防具全般。
・住は建物全般の建造とメンテナンス。

領主代行
・領を治める村長みたいなの
・肉体労働は少ないが、中間管理職みたいなもので一番大変ｗ

# 中央出張所
以下の機能を持つ
・出産前後の待機と出産処理
・病気や疾病の治療

# お金
・出来高制で給金される
・貧民エリア内では物々交換なのでほとんど価値がない

# 住まい
・一家族一家
・家のつくりや広さは領内の住担当の総意やら気分やら
・交渉や物々交換次第では作り直してもらうこともできる

# 家族
・所帯持ちも独身もいる

# 治安
・騎兵隊が幅を利かせているので治安は良い
・家も戸締まりしなくていいし、金も保管したまま
・夜は活動停止になりがちで、無闇にうろつくと騎兵隊に問われやすい
